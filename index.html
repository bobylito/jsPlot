plot.js

plot.js is a library to create plots of functions by Alexandre Stanislawski (@bobylito) under MIT licence. The code can be found on github and the documentation is there.

Key features :

  • Canvas based
  • No other dependencies (no jQuery needed for instance)
  • A unique function to plot all the graphs you want

Samples : 

jsPlot(
  "plotHere", 
  {
    Xmin : -5,
    Xmax : 5,
    Ymin : -5,
    Ymax : 5,
    canvasHeight : 250,
    canvasWidth : 250
  },
  [function x(x){return x}, 
  function x2(x){return x*x}]);

jsPlot(
  "plotThere", 
  {
    Xmin : -5,
    Xmax : 5,
    Ymin : -5,
    Ymax : 5,
    canvasHeight : 250,
    canvasWidth : 250
  },
  [function x3(x){return x*x*x},
  function q(x){return 1/x}]);
